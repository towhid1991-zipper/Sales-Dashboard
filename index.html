Setup & Connection

Connect this dashboard to a shared Google Sheet so all colleagues see the same live data.

Step-by-step Google Sheets connection

1
Create a new Google Sheet. Rename the first tab to Sales. In row 1, add these exact headers in order:
PI Date | Proforma Invoice No | Customer Name | Buyer Name | Item Code | Generic Item | Style Reference | PI Value | Factory | Sales Person Name
2
Go to script.google.com → New Project. Delete the default code and paste the Apps Script code (provided separately as apps-script.gs). Click the disk icon to save.
3
In the script editor, find the line const SHEET_ID = "PASTE_YOUR_SHEET_ID_HERE"; and replace it with your sheet's ID (the long string in your Sheet's URL between /d/ and /edit).
4
Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and authorize permissions when prompted.
5
Copy the Web app URL you get (ends in /exec) and paste it below, then click Connect.

How colleagues use this

Once connected and this file is published on GitHub Pages, share the page link with colleagues. The connection URL is saved inside this HTML file itself — so as long as everyone opens the same published page, they're automatically connected to the same shared sheet. No setup needed on their end.

Tip: After connecting here, re-export this HTML (it auto-saves the URL in your browser) — or simply hardcode the URL into index.html before publishing so it's baked in for everyone.
